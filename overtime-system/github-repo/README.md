# Overtime System (OT-SYS)

Aplikasi manajemen lembur karyawan berbasis Google Sheets + Google Apps Script.

- **Frontend**: GitHub Pages (folder `/docs`)
- **Backend**: Google Apps Script Web App
- **Database**: Google Spreadsheet

## Cara akses
Buka: `https://USERNAME.github.io/overtime-system/`

Backend URL diatur via tombol "⚙ Atur koneksi backend" di halaman login,
atau tertanam di `docs/index.html` (konstanta `GAS_API_URL_DEFAULT`).
